[ Javascript ] 타입, 값, 변수 기초 정리

2017-01-18 Javascript 기초



타입, 값, 변수

  • 데이터 타입

    • 프로그래밍 언어로 다룰 수 있는 값의 유형

  • 변수

    • 변수는 var 키워드로 선언한다.
    • 값을 저장하며 그 이름을 통하면 값을 참조 할 수 있다.

자바스크립트

  • 타입

    • 원시타입(primitive type)
      • 숫자
      • 문자열
      • 불리언
      • null
      • undefined
    • 객체타입(object type)
      • 위 외의 값이 객체

      • 이름과 값을 갖는 프로퍼티의 집합

        • 숫자, 문자열, 객체 등등

      • 순서가 없는 값들의 집합

      • 각 값에는 이름이 지정 되어있슴.

      • 배열 객체

        • 순서가 있는 값들의 집합
        • 각 값에는 번호가 부여된다.

      • 함수 객체

        • 실행코드를 가지고 있는 객체
        • 함수를 호출하면 코드가 수행되고 계산된 값을 반환한다.
        • 생성자
          • 새로 생성된 객체를 초기화 하기 위해 사용되는 함수
          • 객체들의 클래스를 정의한다.

  • 코어 자바스크립트 클래스

    • Array 클래스

      JavaScript Array 객체는 배열 생성에 사용되는 전역 객체입니다. 상위 수준의 목록과 같은 객체입니다.

      참조https://developer.mozilla.org/.../Array
    • Function 클래스

      Function 생성자는 새로운 Function 객체를 생성합니다. JavaScript에서 모든 함수는 실제로 Function 객체입니다.

      참조https://developer.mozilla.org/.../function
    • Date 클래스

      단일 순간을 나타내는 JavaScript Date 인스턴스를 만듭니다. Date 개체는 UTC 1970 년 1 월 1 일 이후의 밀리 초 수인 시간 값을 기반으로합니다.

      • 인수가 제공되지 않으면 생성자는 시스템 설정에 따라 현재 날짜와 시간에 대한 JavaScript Date 객체를 만듭니다.
      • 적어도 두 개의 인수가 제공되면 누락 된 인수는 1 (하루가 누락 된 경우) 또는 다른 모든 인수가 0으로 설정됩니다.
      • JavaScript 날짜는 UTC 1970 년 1 월 1 일 자정 이후의 시간 값 (밀리 초)을 기준으로합니다. 하루는 86,400,000 밀리 초를 보유합니다.
      • JavaScript Date 객체는 현지 시간 메서드뿐만 아니라 여러 가지 UTC (범용) 메서드를 지원합니다. 그리니치 표준시 (GMT)라고도하는 UTC는 세계 시간 표준에 따라 설정된 시간을 나타냅니다. 현지 시간은 JavaScript가 실행되는 컴퓨터에 알려진 시간입니다.
      • JavaScript Date를 함수로 호출하면 (즉, new 연산자가없는 경우) 현재 날짜와 시간을 나타내는 문자열이 반환됩니다.
      참조https://developer.mozilla.org/.../Date

    • Error 클래스

      Error 생성자는 오류 객체를 만듭니다. Error 객체의 인스턴스는 런타임 오류가 발생할 때 발생합니다. Error 객체는 사용자 정의 예외의 기본 객체로 사용할 수도 있습니다.

      • 런타임 오류로 인해 새로운 Error 객체가 생성되어 throw됩니다.
      • 이 페이지는 Error 객체 자체의 사용과 생성자 함수로서의 그것의 사용을 문서화합니다. Error 인스턴스에서 상속되는 속성 및 메서드 목록을 보려면 Error.prototype을 참조하십시오.
      참조https://developer.mozilla.org/.../Error

  • 자바스크립트 인터프리터

    • 메모리 관리를 위해 자동으로 관리를 위해 자동으로 가비지 컬렉터를 수행한다.

  • 자바스크립트는 객체 지향언어이다.

    • 다양한 타입의 값을 다루는 전역 함수를 정의해 두기보다. (C언어…?)
      • 어떤 값과 작동하는 메서드를 그 값의 타입에 정의해둔다는 말이다.
    • 예를들어 배열의 원소들을 정렬할 때는 배열 a를 sort() 함수에 인자로 전달하지 않는다.
      • 대신 a의 sort() 메서드를 호출하여 정렬한다.
        • a.sort(); // sort(a)의 객체 지향 버전
    • 일반적으로 객체만 메서드를 소유한다.
    • 하지만 숫자, 문자열, 불리언 값은 메서드를 가진것 처럼 작동한다.
    • null, undefined는 자바스크립트에서 유일하게 메서드를 가질 수 없는 값이다.

  • 자바스크립트 타입 구분하기

    • 원시타입, 객체타입
    • 메서드를 가진 타입, 못가진 타입
    • 수정가능한 타입, 수정할수 없는 타입
      • 수정가능(mutable)
        • Object
        • Array
      • 수정불가(immutable)
        • 숫자
        • 불리언
        • null
        • undefined
        • 문자열
          • 문자열은 수정가능한것처럼 생각할 수 있지만 자바스크립트에서 지원하지 않는다.

  • 자바스크립트 값의 타입변환

    • 자바스크립트 변수들은 타입이 정해져 있지 않다.
    • 같은 변수에 여러 타입의 값을 여러번 할당 할 수있다.
    • 또 할당된 값의 타입이 다른 곳에 사용할때 값이 바뀌는 규칙에따라 적절하게 타입을 변환해 사용한다.

  • 어휘 유효범휘

    • 함수 밖에 선언된 변수들은 전역변수이고 자바스크립트 어디서든 사용할 수 있다.
    • 함수 안에 선언된 변수들은 유효범위가 함수 영역이며 오직 함수 안에서만 사용할 수있다.
