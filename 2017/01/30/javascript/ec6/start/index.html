ECMAScript

  • ECMA이란?

    • W3C 같은 표준화 단체이다.

  • ECMAScript란?

    • ECMAScript는 ECMA가 javasciprt를 표준화한 것
    • ECMAScript는 브라우저에 한정되지 않은 JS의 일반적인 측면을 다루고 표준화 한다.

  • ECMAScript이 정의 하는 것

    • 언어의 문법 – 파싱(parsing) 규칙, 키워드, 구문, 선언법, 연산자 등.
    • 타입 – boolean, number, string, object 등.
    • 프로토타입(prototype)과 상속(inheritance)
    • 빌트인(built-in) 객체와 함수로 구성된 표준 라이브러리 – JSON, Math, Array 메소드, Object 메소드 등.

  • ECMAScript 6란?

    • 2015년 4월 23일에 ES5가 나온 이후 6년만에 나온 Javascript 프로그래밍 언어의 새로운 표준이다.
    • ES6는 수년에 걸친 협업의 산물이다.
    • ES6가 제시하는 새로운 언어적 개념과 라이브러리의 기능은 보석과 같다.
    • JS언어에 있었던 업그레이드 중에서 가장 혁신적이다.
    • ES6가 동비하는 새로운 요소는 화살표 함수와 간편한 문자열 치환같은 가벼운 편의기능부터, 프록시와 제너레이터 같은 머리 아픈 새로운 개념까지 다양하게 제공한다.
