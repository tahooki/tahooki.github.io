레스트 파라메터(rest parameter)

  • API 정의 variadic function(가변 함수)가 필요한 경우

    • arguments 방식

      • rguments 객체는 함수에 전달된 파라메터들을 담고 있는 배열

      • 위 상황에서 단점

        • 여러개의 인자를 취한다는 사실을 한번에 알아보기 어렵다
        • arguments 객체를 순회할 때 인덱스가 0이 아니라 1에서 시작하도록 주의해야 한다.

    • 레스트 파라메터(rest parameter)

      • 나머지 파라메터들을 배열로 받는 파라메터

      • 만약 추가가 없다면 빈 배열로 가지게 된다.

        • 레스트 파라메터 변수 값이 undefined가 되는 일은 없다.

디폴트 파라메터(default parameter)

  • 기본 값을 넣어주는 파라메터

    • animalSentence()라고 호출하면
      • “Lions and tigers and bears! Oh my!”가 리턴
    • animalSentence(“elephants”)라고 호출하면
      • “Lions and elephants and bears! Oh my!”가 리턴
    • animalSentence(“elephants”, “whales”)라고 호출하면
      • “Lions and elephants and whales! Oh my!”가 리턴

  • python과 달리, 디폴드 값은 함수를 호출하는 시점에 계산된다.

    • animalSentenceFancy(“bears”)라고 호출할 때
      • “Lions and bears and sealions. Oh my!”가 리턴된다.
    • undefiend를 전달
      • animalSentence(undefined, “unicorns”)라고 호출하면
        • “Lions and tigers and unicorns! Oh my!”가 리턴
